现在有一个需求就是要把筛选出其中的成年人，放到一个新的集合里，传统方式应该怎么做，遍历集合，判断

1
2
3
4
5
6

List<People> adults = new ArrayList<>();
        for(People peo : people){
            if(peo.getAge() > 18){
                adults.add(peo);
            }
        }

这种方式虽然直观，但是当情况变地负责时，代码会变得复杂。

使用Stream API来实现一下

1
2
3

List<People> adults = people.stream()
                .filter(peo -> peo.getAge() > 18)
                .collect(Collectors.toList());

Stream Api地引入为数据集合的操作提供了一种新的范式，Stream本身并不是数据结构，不会存储数据或者改变数据源，它仅定义处理方式，可以视为一种高级迭代器，不仅能够支持顺序处理，还能进行并行处理，为集合的过滤、映射、聚等操作提供了一种高效的实现方式。

Stream流的3个核心步骤

第一步：创建流Stream Creation

可以从多种数据源创建，集合，数组，IO等

第二步：中间操作 Intermediate Operations

第三步：终端操作 Terminal Operations

HashMap<String, List<People>> collect = people.stream()
                .collect(Collector.of(
                        HashMap::new,
                        (map, person) -> {
                            System.out.println("combiner: " + person);
                            map.computeIfAbsent(person.getCountry(), k -> new ArrayList<>()).add(person);
                        },
                        (left, right) -> {
                            right.forEach((key, value) -> left.merge(key, value, (list, newList) -> {
                                list.addAll(newList);
                                return list;
                            }));
                            return left;
