利用hugo和githubpages创建个人博客

1.hugo安装

点击进入官网下载

选择对应版本，以64位windows版本为例

解压后得到hugo.exe文件

找一个安装目录，例如D:\code\hugo

在hugo目录下新建一个新的文件夹bin,将上面得到的hugo.exe文件复制到bin文件夹里

将bin所在目录放到系统环境变量的Path里

运行cmd

1

hugo version

显示如图所示即为安装成功

2.hugo使用

选择合适的目录，建立博客，这里名称为xiaoyume

1

hugo new site xiaoyume

自动生成test文件夹以及其下的一堆子文件，博客初始化成功

1

cd xiaoyume

进入到xiaoyume目录下面

下载一个hugo主题，用以网站页面显示，这里选择even主题

1

git clone https://github.com/olOwOlo/hugo-theme-even themes/even

在themes目录下出现了一个even目录，这就是主题资源

在even/exampleSite目录下有一个config.toml

复制其中所有内容到test目录下面的同名config.toml里，这个就是配置文件，用以控制页面效果

3.新建md文件，博客内容所在

使用下面命令新建一个md文件，默认生成在content目录下，请加上post文件夹

1

hugo new post/test.md

生成的md文件会带有如图所示的文件信息，draft表示是否是草稿，不显示，请改为false

在md文件中输入文档信息，这些内容后续会显示在网页上

1

hugo server

运行后会在本地运行服务

运行成功后如上图显示，在浏览器中输入http://localhost:1313可以访问本地网页信息

注意在运行时可能会报错，跟版本有关，如果出现错误，请删除themes/even/layouts/partials/head.html文件里

{{- template “_internal/google_news.html” . -}}这行代码

4.配置githubpages

githup上新建仓库，名称为用户名.github.io,注意勾选上readme.md，这里是为了创建main分支

找一个合适的目录把这个仓库clone到本地

例如目录是D:\code\git\xiaoyume.github.io

因为使用githubpages托管网页需要先使用hugo生成静态页面，hugo执行后生成的文件在博客目录里的public文件夹内，到时候需要把这个目录的文件全部push到githubpage的仓库里，所以这里可以对public文件夹和本地库做一个软链接，需要先把public文件夹删除

win11使用桌面右键win图标，**终端（管理员）**模式，注意默认启动的是powershell，需要在里面切换cmd

1

mklink /j D:\code\git\xiaoyume\public D:\code\git\xiaoyume.github.io

链接完成后执行hugo生成的文件会直接到本地库里

在博客目录xiaoyume下面，依次执行

1
2
3
4
5

hugo
cd public
git add .
git commit -m "auto"
git push

不出意外的话，push到远程库后等待githubpages完成部署，就能通过链接xiaoyume.github.io访问到博客了

到此就结束了，至于主题页面的设置的话，后面再结束
